## ‚ö° Quick Start

### 1. Install Dependencies

```bash
npm install
```

### 2. Add Images (IMPORTANT!)

**Required:** Replace placeholder images in `public/images/`:

- **missing-person.jpg** - Photo from: https://share.google/hXgIhso8QPIaRLxye
- **og-image.jpg** - Social media preview (1200x630px)

See [IMAGE_INSTRUCTIONS.md](./IMAGE_INSTRUCTIONS.md) for detailed steps.

### 3. Configure Environment

```bash
cp .env.local.example .env.local
# Edit .env.local with your actual values
```

### 4. Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### 5. Build for Production

```bash
npm run build
npm start
```

## üöÄ Deployment

### Option 1: Vercel (Recommended - Free & Easy)

1. Push code to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project" and import your repository
4. Add environment variables from `.env.local`
5. Click "Deploy"

**Your site will be live in ~2 minutes!**

### Option 2: Netlify

1. Push to GitHub
2. Go to [netlify.com](https://netlify.com)
3. New site from Git ‚Üí Select repository
4. Build command: `npm run build`
5. Publish directory: `.next`
6. Deploy

### Option 3: Custom Server

```bash
npm run build
npm start
```

For production, use PM2:
```bash
npm install -g pm2
pm2 start npm --name "missing-website" -- start
```
